此目录用于放置功能模块 Lua 文件。

当前已包含：

- `metrics.lua`：战斗指标数据模型与汇总逻辑
- `metrics_collector.lua`：战斗事件采集器（Combat Log / 怒气 / Buff）
- `metrics_decision.lua`：决策引擎（姿态导向 + 下个技能 + 英勇/顺劈建议）
- `metrics_icon.lua`：决策提示图标（技能推荐可视化）
- `metrics_panel.lua`：决策调试面板（`/fury metrics`）
- `minimap.lua`：小地图图标
- `options.lua`：设置页
- `slash.lua`：命令入口
